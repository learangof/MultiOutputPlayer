# MultiOutputPlayer

This is an Media Player with multiple audio outputs

Things you will use:
* [Node.js](https://nodejs.org)
* [NW.js](https://nwjs.io)
* [Bulma](https://bulma.io/documentation)
* [TypeScript]
* JQuery
* Gulp

### To Install
``` js
// In the root folder
npm install
// In the src folder
```
